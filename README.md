# Sistema de Análisis de Ventas — Proyecto Integrador

Proyecto end-to-end que combina SQL Server, Python y Power BI 
para análisis completo de ventas, vendedores y rentabilidad.

## Tecnologías
- SQL Server (7 tablas, 2 views, consultas avanzadas)
- Power BI Desktop (DAX, dashboard de 3 páginas)
- Python (pandas, matplotlib, openpyxl)

## Contenido
- `integrador_analytics.sql` — Base de datos completa
- `dashboard-integrador.pbix` — Dashboard ejecutivo de 3 páginas
- `reporte_integrador.py` — Script de análisis y reporte automático
- `reporte_integrador.png` — Dashboard generado con Python

## Dashboard Power BI
- Página 1: Resumen Ejecutivo (KPIs + evolución mensual)
- Página 2: Análisis de Ventas (productos, categorías, segmentos)
- Página 3: Rendimiento de Vendedores
